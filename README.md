# Suman Suiwal — Portfolio Website

A responsive personal portfolio website built as part of the internSpark Frontend Development Internship (Task 1).

## 🔗 Live Demo
https://sumansuiwal992.github.io/portfolio-website/

## 📌 About
This is a mobile-friendly personal portfolio website with Home, About, Skills, Project, and Contact sections, featuring smooth scroll navigation and a dark theme design.

## ✨ Features
- Fully responsive layout (mobile, tablet, desktop)
- Fixed navbar with smooth scroll navigation
- Mobile hamburger menu
- Dark theme UI
- Hero section with profile photo
- About section with highlight stats
- Skills section with progress bars
- Project showcase with live demo & GitHub links
- Contact form with success message
- Footer with LinkedIn and GitHub icons

## 🛠️ Built With
- HTML5
- CSS3
- Bootstrap 5.3.3
- JavaScript (ES6+)
- Bootstrap Icons
- Google Fonts (Poppins, Fira Code)
